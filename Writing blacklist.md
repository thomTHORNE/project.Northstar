# Writing blacklist

Banned constructions. This extends the tone rules in [CLAUDE.md](CLAUDE.md) → *Tone and writing style* with a concrete, growing list of filler and jargon to avoid. Check prose against it before committing, and add a new row whenever another offender is caught.

The point is not the exact words in the left column — it is the register they belong to: editorializing lead-ins, colloquial metaphors, and jargon standing in for a plain word.

| Avoid | Use instead |
|---|---|
| "the striking thing is…", "the interesting thing is…" | State the point directly; cut the lead-in. |
| "earns its keep", "earns its place" | "is needed for", "does". |
| "bite" (meaning apply / matter) | "apply", "matter". |
| "red herring" | "not the deciding factor", "beside the point". |
| "an island" (e.g. "each install is an island") | "self-contained", "separate". |
| "cuts the other way" | "the reverse is also true". |
| "pulls toward", "what genuinely pulls toward" | "needs", "argues for". |
| "survive contact with [X]'s reality" | "hold up", "apply". |
| "delta" (meaning difference) | "difference". |
| "luxury", "a later luxury" (meaning optional) | "optional", "can wait", "deferrable". |
| "slots in cleanly", "slots in" | "can be added", "fits". |
| "held line by line against" | "compared line by line against". |
